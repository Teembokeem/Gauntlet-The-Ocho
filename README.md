#GAUNTLET THE OCHO
##Turn-based RPG
Gauntlet The Ocho is an interactive game that pits you, a GA student, against your beloved Cohort 8 instructors; "EZmode" Ezra, "Good Guy" Jim, and "MasterMind" Phil.
